# VBS.LoveLetterとは？VBScript添付ファイルで広がるワーム【SG試験】

## まず結論

VBS.LoveLetterは、**VBScriptで書かれた添付ファイルを実行させ、メールなどで感染を広げるワーム型マルウェア**です。

SG試験では、Wanna Cryptorのようなファイル暗号化・金銭要求ではなく、**「VBScript」「添付ファイル」「メールで拡散」**を手掛かりに判断します。

## このページで切り分けること（先にここだけ）

このページは、**VBS.LoveLetter**を中心に整理します。

- VBS.LoveLetter：VBScriptの添付ファイルを実行させ、メールなどで拡散する
- Bagleワーム：メールアドレス収集や複数経路での拡散が中心
- Wanna Cryptor：ファイル暗号化と金銭要求が中心

> 迷ったら、**「VBScript添付ファイルの話か」** を見ます。

## 直感的な説明

VBS.LoveLetterは、受信者に「知っている相手から届いたラブレターの添付ファイル」のように見せて、ファイルを開かせるマルウェアです。

利用者が添付ファイルを実行すると、PCが感染し、さらにアドレス帳などを使って別の人へ同じようなメールを送ります。

受け取った人も、知人から来たように見えるため、添付ファイルを開いてしまう可能性があります。

SG試験では、VBS.LoveLetterを細かいプログラムとして覚えるよりも、**添付ファイルを開かせる心理的な誘導と、メールによる感染拡大**として考えると分かりやすくなります。

## 定義・仕組み

VBS.LoveLetterは、Love Letterウイルス、I LOVE YOUウイルスなどとも呼ばれるマルウェアです。

基本の流れは、次のように整理できます。

1. 利用者にVBScriptファイルが添付されたメールが届く
2. 利用者が添付ファイルを実行する
3. PCが感染し、ファイルの書換えや情報送信などの被害が起こる
4. アドレス帳などを使って、さらに別の宛先へメールを送る
5. 受信者が添付ファイルを開くと感染が広がる

ここで大事なのは、**VBScriptの添付ファイルを実行させること**と、**メールなどで拡散すること**です。

JPCERT/CCの解説でも、Love LetterはVBScriptで書かれたウイルスで、電子メールやIRCなど複数の方法を使って感染を試みたと説明されています。詳しく確認したい場合は、JPCERT/CCの[インターネットセキュリティの歴史 第7回「Love Letter ウイルス」](https://www.jpcert.or.jp/tips/2007/wr073001.html)が参考になります。

SG試験では、VBScriptの文法を問うというより、次の判断が重要です。

- 添付ファイルを安易に実行しない教育ができているか
- 不審な拡張子を確認できるか
- メールフィルタやマルウェア対策ソフトを使っているか
- 感染端末を隔離し、二次感染を止められるか
- 知人からのメールでも添付ファイルを無条件に信頼しないか

## どんな場面で使う？

VBS.LoveLetterの知識は、メール添付ファイルによるマルウェア感染や、拡張子の確認を問う問題で使います。

特に、次のような場面で出題されやすいです。

- `LOVE-LETTER-FOR-YOU.TXT.VBS` のような添付ファイルを実行した
- VBScriptファイルを開いた利用者のPCが感染した
- アドレス帳の宛先へメールが送信された
- メールで感染が広がった
- 添付ファイルの拡張子確認や利用者教育が対策になる

実務では、VBS.LoveLetterのようなマルウェアに対して、技術対策だけでなく、利用者教育が重要です。

たとえば、次のような対策が考えられます。

- 不審な添付ファイルを開かない
- 拡張子を確認する
- 実行形式やスクリプトファイルをメールで受け取ったときは警戒する
- マルウェア対策ソフトを最新に保つ
- 感染が疑われる端末をネットワークから切り離す

## よくある誤解・混同

VBS.LoveLetterは、ほかのマルウェア名と混同しやすいです。

| 用語 | 判断ポイント |
|---|---|
| VBS.LoveLetter | VBScriptの添付ファイルを実行させ、メールなどで拡散する |
| Bagleワーム | 電子メールやファイル共有を使って感染を広げ、メールアドレスを収集する |
| Wanna Cryptor | SMBv1の脆弱性を悪用し、ファイルを暗号化して金銭を要求する |
| SQL Slammer | SQL Serverの脆弱性を悪用し、大量通信でネットワークに影響を与える |
| マクロウイルス | WordやExcelなどのOffice文書のマクロを悪用する |

SG試験では、次のように切り分けます。

**VBS.LoveLetterと判断する材料**

- VBScript
- `.vbs` ファイル
- 添付ファイルの実行
- メールによる拡散
- Love Letter / I LOVE YOU

**VBS.LoveLetterではないと判断する材料**

- メールアドレス収集や複数経路での拡散が強調されるなら、Bagleワームを疑う
- SQL Serverの脆弱性と大量通信が中心なら、SQL Slammerを疑う
- ファイル暗号化と金銭要求が中心なら、Wanna Cryptorを疑う
- WordやExcelのマクロが中心なら、マクロウイルスを疑う

選択肢では「メールで広がる」とだけ書かれていると、Bagleワームと似て見えます。  
そのため、**VBS.LoveLetterは「VBScript添付ファイル」を強い判断材料にする**のがポイントです。

## 確認問題（SG試験対策）

メール添付のスクリプトファイルを利用者が開いたことで感染し、端末内の連絡先を使って同様のメールを広げた。この特徴に最も近いマルウェアはどれか。

- ア. ルートキット
- イ. VBS.LoveLetter
- ウ. SQL Slammer
- エ. アドウェア

**正解：イ**

### 解説

- ア：不適切。ルートキットは不正な存在や権限を隠す仕組みです。
- イ：適切。VBScriptなどのスクリプト添付ファイルとメール拡散が判断材料です。
- ウ：不適切。SQL SlammerはSQL Serverを狙うワームです。
- エ：不適切。アドウェアは広告表示が中心です。

👉 判断ポイント

添付スクリプトを開かせ、メールで広がるならVBS.LoveLetterを疑います。

## まとめ（試験直前用）

VBS.LoveLetterは、VBScriptの添付ファイルを実行させて感染するワーム型マルウェアです。

メールなどを使って感染を広げる点が特徴です。

SG試験では、VBScript、`.vbs`、添付ファイル、メール拡散が出たらVBS.LoveLetterを疑います。

メール拡散だけならBagleワームとも迷いますが、VBScript添付ファイルが強調されていればVBS.LoveLetterと切り分けます。
