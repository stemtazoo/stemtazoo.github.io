# BSSIDとは｜SG試験で押さえるSSIDとの違い

## まず結論

**BSSID** は、無線LANで**アクセスポイントを識別するための情報**です。

SG試験では、次のように切り分けると分かりやすいです。

```text
SSID  ：無線LANのネットワーク名
BSSID ：アクセスポイントを識別する情報
```

スマートフォンやPCのWi-Fi一覧に表示される名前は、基本的にSSIDです。
一方、BSSIDは利用者が普段意識する名前ではなく、アクセスポイントごとの識別に使われます。

---

## 直感的な説明

SSIDを「お店の看板」と考えると、BSSIDは「その店舗そのものの番号」のようなイメージです。

たとえば、同じ会社の建物に、同じSSIDを設定したアクセスポイントが複数あるとします。

```text
SSID：Company-WiFi

アクセスポイントA：BSSID A
アクセスポイントB：BSSID B
アクセスポイントC：BSSID C
```

利用者から見ると、Wi-Fi名は同じ `Company-WiFi` に見えます。

しかし、実際には通信しているアクセスポイントは1台ずつ異なります。
そのアクセスポイントを区別するための識別情報がBSSIDです。

---

## 定義・仕組み

BSSIDは、**Basic Service Set Identifier** の略です。

無線LANにおいて、基本的にはアクセスポイントを識別するために使われます。
多くの場合、アクセスポイントの無線インタフェースのMACアドレスに対応する形で扱われます。

| 項目 | 内容 |
|---|---|
| 正式名称 | Basic Service Set Identifier |
| 役割 | アクセスポイントを識別する |
| 関連する情報 | MACアドレス |
| よく混同する用語 | SSID |
| 試験での見方 | 「アクセスポイント側の識別情報」と考える |

### SSIDとBSSIDの違い

SG試験では、SSIDとBSSIDの違いがよく狙われます。

| 用語 | 何を識別するか | イメージ |
|---|---|---|
| SSID | 無線LANのネットワーク | Wi-Fi名 |
| BSSID | アクセスポイント | アクセスポイントごとの識別情報 |

同じSSIDを複数のアクセスポイントに設定することがあります。
この場合、SSIDは同じでも、BSSIDはアクセスポイントごとに異なります。

```text
SSIDは同じ
BSSIDはアクセスポイントごとに違う
```

ここが一番大事な切り分けポイントです。

---

## どんな場面で使う？

BSSIDは、無線LANの通信で、どのアクセスポイントと接続しているかを識別する場面で使われます。

たとえば、次のような場面です。

- 同じSSIDを持つ複数のアクセスポイントを区別する
- 端末が接続するアクセスポイントを判別する
- 無線LANの調査やトラブル対応で接続先を確認する
- ローミング時に、どのアクセスポイントへ切り替わったかを確認する

会社や学校のように広い場所では、1台のアクセスポイントだけでは電波が届きにくいことがあります。
そのため、同じSSIDを複数のアクセスポイントに設定して、移動しても接続しやすくします。

このとき、利用者には同じWi-Fi名に見えていても、内部的にはBSSIDでアクセスポイントを区別しています。

---

## よくある誤解・混同

### 誤解1：BSSIDはWi-Fi一覧に表示される名前である

Wi-Fi一覧に表示される名前は、基本的にSSIDです。

BSSIDは、利用者が普段選ぶ名前ではありません。
アクセスポイントを識別するための情報です。

```text
Wi-Fi一覧の名前 → SSID
アクセスポイント識別 → BSSID
```

### 誤解2：SSIDとBSSIDは同じ意味である

SSIDとBSSIDは似ていますが、意味は違います。

SSIDはネットワーク名です。
BSSIDはアクセスポイントを区別するための情報です。

試験では、

> 「最大32オクテットのネットワーク識別子」

とあればSSID、

> 「アクセスポイントのMACアドレスと関係する識別情報」

とあればBSSIDを疑います。

### 誤解3：SSIDが同じならBSSIDも同じである

SSIDが同じでも、BSSIDが同じとは限りません。

複数のアクセスポイントに同じSSIDを設定することがあるためです。

```text
同じSSIDでも、アクセスポイントが違えばBSSIDは違う
```

この考え方は、選択肢を切るときに役立ちます。

---

## 似た用語との切り分け

| 用語 | 試験での判断ポイント |
|---|---|
| SSID | 無線LANのネットワーク名 |
| BSSID | アクセスポイントを識別する情報 |
| ESSID | 複数のアクセスポイントを含む無線LANの識別名として使われることがある |
| MACアドレス | ネットワーク機器ごとの物理アドレス |
| IPアドレス | ネットワーク上の通信先を示す論理アドレス |

特に、SSIDとBSSIDは名前が似ているため混同しやすいです。

SG試験では、次のように読むと判断しやすくなります。

```text
ネットワーク名・最大32オクテット → SSID
アクセスポイント・MACアドレス → BSSID
```

---

## 試験直前まとめ

BSSIDは、SG試験では次の3点で押さえます。

- **BSSIDは、アクセスポイントを識別する情報**
- **SSIDは、無線LANのネットワーク名**
- **同じSSIDでも、BSSIDはアクセスポイントごとに異なることがある**

ひっかけで狙われやすいのは、SSIDとの混同です。

```text
SSID  → Wi-Fi名
BSSID → アクセスポイント識別
```

この対応で覚えると、選択肢をかなり切りやすくなります。
