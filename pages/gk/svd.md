## まず結論

SVD（Singular Value Decomposition：特異値分解）とは、**行列を3つの行列に分解し、データの重要な構造を取り出す線形代数の手法**です。

G検定では、特に **PCAとの関係** や **次元削減に使われる点** が問われます。

| 観点 | SVDのポイント |
|---|---|
| 何をする？ | 行列を分解する |
| 何が分かる？ | データの重要な方向や構造 |
| 何に使う？ | 次元削減、ノイズ除去、特徴抽出 |
| 関係が深い用語 | PCA、LSA、行列分解 |
| G検定キーワード | 特異値、次元削減、PCAの計算 |

問題文に **「行列分解」「特異値」「次元削減」「PCAの計算」** が出てきたら、SVDを疑います。

---

## 直感的な説明

SVDは、ひとことで言うと、

> **データを“重要な成分”に分解して、要点だけを取り出す方法**

です。

たとえば、大きな表データを考えます。

行にはユーザー、列には商品が並んでいて、各セルに評価点が入っているとします。

この表には、細かいノイズもありますが、全体としては

- 映画好きの傾向
- アニメ好きの傾向
- 高評価しやすいユーザー
- 人気商品のまとまり

のようなパターンが隠れているかもしれません。

SVDは、このような大きな行列を分解して、重要なパターンを取り出します。

重要度の低い成分を捨てると、データを小さく圧縮できます。  
これが次元削減につながります。

---

## 定義・仕組み

SVDでは、行列を3つの行列に分解します。

細かい式を暗記する必要はありません。  
G検定では、次のイメージで十分です。

```text
元の行列
↓
左側の特徴
×
重要度
×
右側の特徴
```

この「重要度」にあたるものが **特異値** です。

### 特異値とは

特異値は、各成分がどれくらい重要かを表す値です。

大きい特異値ほど、データの中で重要な構造を表します。

| 用語 | 意味 |
|---|---|
| 左特異ベクトル | 行方向の特徴を表す |
| 特異値 | 成分の重要度を表す |
| 右特異ベクトル | 列方向の特徴を表す |

### 次元削減に使える理由

SVDでは、特異値の大きい成分を残し、小さい成分を捨てることができます。

これにより、

- 重要な構造を残す
- ノイズを減らす
- データ量を小さくする
- 特徴量を少なくする

といったことができます。

つまり、SVDは **行列を分解して重要な成分だけを残すことで、次元削減に使える** 手法です。

---

## いつ使う？（得意・不得意）

### 得意な場面

SVDは、行列データの構造を取り出したい場面で使われます。

例：

- 次元削減
- ノイズ除去
- 画像圧縮
- 推薦システム
- 潜在意味解析（LSA）
- PCAの計算

データの中にある主要なパターンを取り出すために使われます。

### PCAとの関係

G検定では、SVDとPCAの関係がよく問われます。

PCAは、データの分散が大きい方向を見つけて次元削減する手法です。

SVDは、そのPCAを計算するために使われることがあります。

| 用語 | 目的 | 試験での見分け方 |
|---|---|---|
| PCA | 分散が大きい方向へ射影して次元削減する | 分散最大化、主成分 |
| SVD | 行列を分解して重要成分を取り出す | 特異値、行列分解 |
| 固有値分解 | 正方行列を固有値・固有ベクトルで分解する | 固有値、固有ベクトル |

PCAは「何をしたいか」、SVDは「その計算に使える行列分解」と考えると整理しやすいです。

### 注意が必要な場面

SVDは線形な手法です。

そのため、複雑な非線形構造をうまく表せない場合があります。

| 目的 | 疑う用語 |
|---|---|
| 線形な次元削減 | PCA / SVD |
| 非線形な次元削減 | t-SNE / UMAP |
| 分類したい | 教師あり学習 |
| クラスタに分けたい | クラスタリング |

G検定では、SVDを「教師あり学習の分類器」と考えないように注意します。

---

## G検定ひっかけポイント

### 誤解1：SVDは教師あり学習の分類アルゴリズム

これは誤りです。

SVDは、行列を分解してデータの構造を取り出す線形代数の手法です。

ラベル付きデータを使って分類器を学習する手法ではありません。

### 誤解2：SVDは非線形次元削減である

これも誤りです。

SVDは線形な行列分解です。

非線形な構造を可視化・次元削減したい場合は、t-SNEやUMAPなどの文脈になります。

### 誤解3：SVDとPCAは完全に同じ

SVDとPCAは関係が深いですが、完全に同じ言葉ではありません。

PCAは、分散が大きい方向を見つけて次元削減する手法です。  
SVDは、行列を分解する計算方法です。

PCAの計算にSVDが使われることがあります。

### 誤解4：特異値はクラス分類のラベルである

特異値はラベルではありません。

特異値は、分解された成分の重要度を表します。

### 選択肢の切り方

| 問題文の表現 | 判断 |
|---|---|
| 行列を3つに分解する | SVD |
| 特異値が重要度を表す | SVD |
| 分散最大化 | PCA |
| 固有値・固有ベクトル | 固有値分解 |
| 非線形な可視化 | t-SNE / UMAP |
| 教師あり分類 | SVM、決定木など |

**「行列分解 + 特異値 + 次元削減」ならSVD** と覚えると切りやすいです。

---

## まとめ（試験直前用）

- SVD = 行列を3つの行列に分解する手法
- 特異値は成分の重要度を表す
- 大きい特異値を残すことで次元削減できる
- PCAの計算に使われることがある
- 教師あり分類や非線形次元削減ではない

**判断基準：特異値を使って行列の重要成分を取り出すなら、SVD。**
