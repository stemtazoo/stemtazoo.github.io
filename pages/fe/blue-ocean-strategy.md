## まず結論

ブルーオーシャン戦略とは、**競争の激しい既存市場を避け、新しい価値と市場をつくる戦略**です。

基本情報技術者試験では、次の切り分けが重要です。

```text
既存市場で他社との違いをつくる
→ 差別化戦略

新しい価値と市場をつくり、競争自体を避ける
→ ブルーオーシャン戦略
```

単に「他社と異なる機能を持つ製品を作る」と書かれているだけでは、ブルーオーシャン戦略とは限りません。

## 直感的な説明

多くの企業が同じ顧客を奪い合う市場を考えます。

```text
同じ顧客
同じ用途
似た製品
価格競争
```

このような競争の激しい市場では、値下げや機能追加で他社と争うことになります。

ブルーオーシャン戦略では、その競争の中で勝とうとするのではなく、別の価値を考えます。

```text
今まで対象でなかった顧客
新しい使い方
新しい組合せ
新しい市場
```

つまり、競争相手と同じ土俵で戦うのではなく、**新しい土俵をつくる**イメージです。

## 定義・仕組み

ブルーオーシャン戦略は、既存の競争ルールから離れ、新しい需要を生み出すことを目指します。

対比されるのが、レッドオーシャンです。

| 市場 | 特徴 | 競争の考え方 |
|---|---|---|
| レッドオーシャン | 既存企業が激しく競争する市場 | 他社より優れた製品や価格で勝つ |
| ブルーオーシャン | 競争が少ない新しい市場 | 新しい価値をつくり、競争を避ける |

ブルーオーシャン戦略では、単なる高機能化だけでなく、顧客にとっての価値を見直します。

```text
不要な機能を減らす
新しい価値を追加する
新しい顧客層を取り込む
価格と価値の組合せを変える
```

このテーマは、基本情報技術者試験の「経営戦略」と関係する内容です。公式の出題範囲やシラバスは、[IPA：基本情報技術者試験](https://www.ipa.go.jp/shiken/kubun/fe.html)から確認できます。

コモディティ化との違いは、[コモディティ化とは？イノベーションやブルーオーシャンとの違い]({{ '/fe/commoditization/' | relative_url }})でも確認できます。

## 科目Aでどう出る？

科目Aでは、説明文から該当する経営戦略を選ぶ問題として出題されます。

ブルーオーシャン戦略を疑う表現は次のとおりです。

- 競争のない新市場を創造する
- 新しい顧客層や需要を開拓する
- 既存の競争ルールから離れる
- 価格競争を避ける
- 新しい価値を提供する

一方、次の表現だけでは不十分です。

```text
他社と異なる機能をつける
高性能な製品を作る
価格を下げる
```

これらは既存市場の中での差別化や価格競争であり、必ずしもブルーオーシャンではありません。

```text
既存市場で優位に立つ
→ 差別化戦略・コストリーダーシップ戦略

競争の少ない新市場をつくる
→ ブルーオーシャン戦略
```

## どんな場面で使う？

市場が成熟し、製品の違いが小さくなると、企業は価格競争に巻き込まれやすくなります。

```text
製品差が小さくなる
↓
価格競争が激しくなる
↓
利益が出にくくなる
```

このような状態から抜け出す方法の一つが、ブルーオーシャン戦略です。

例えば、既存の顧客だけを奪い合うのではなく、これまで利用していなかった人にも使いやすい価値を考えます。

重要なのは、次の関係です。

```text
コモディティ化
→ 差別化しにくくなった市場の状態

ブルーオーシャン戦略
→ 新しい価値と市場をつくる対応策
```

## よくある誤解・混同

| 誤解 | 正しい理解 |
|---|---|
| 他社と異なる機能を付ければブルーオーシャン | 既存市場の中での機能差別化だけでは足りない |
| 高性能な製品を作ればブルーオーシャン | 高性能化は既存市場の競争になることもある |
| 値下げして市場を広げること | 単なる価格競争はレッドオーシャンに近い |
| コモディティ化と同じ意味 | コモディティ化は状態、ブルーオーシャンは戦略 |
| 競争相手が一社もいない必要がある | 重要なのは、新しい価値で競争を小さくすること |

最も大切な切り分けは、**既存市場の中で戦うのか、新しい市場をつくるのか**です。

```text
既存市場で違いを競う
→ 差別化戦略

新しい価値と市場をつくる
→ ブルーオーシャン戦略
```

## まとめ（試験直前用）

- ブルーオーシャン戦略は、競争の少ない新市場をつくる戦略
- 新しい価値や顧客層を生み出し、既存の競争を避ける
- 他社と違う機能を付けるだけでは、必ずしもブルーオーシャンではない
- 既存市場で競うのは差別化戦略、競争の土俵を変えるのがブルーオーシャン
- コモディティ化は市場の状態、ブルーオーシャンはその状態への対応策
