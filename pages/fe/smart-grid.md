## まず結論

スマートグリッドとは、**通信と情報処理技術を使って、発電と電力消費を総合的に制御する電力網**です。

基本情報技術者試験では、次の表現が出たらスマートグリッドを疑います。

```text
電力の供給と需要を
ICTで制御する
↓
再生可能エネルギーの活用
安定した電力供給
最適な需給調整
↓
スマートグリッド
```

ポイントは、単なる発電設備ではなく、**電力網を賢く制御する仕組み**という点です。

## 直感的な説明

スマートグリッドは、電力の流れを見える化し、必要に応じて調整する仕組みです。

```text
発電量を把握する
↓
消費量を把握する
↓
需要と供給を調整する
↓
安定して電力を届ける
```

太陽光発電や風力発電は、天気や時間帯によって発電量が変わります。

そのため、電力の需要と供給をうまく調整する仕組みが必要になります。

スマートグリッドは、その調整をICTで支える仕組みです。

## 定義・仕組み

スマートグリッドは、電力網に通信技術や情報処理技術を組み合わせた仕組みです。

| 目的 | 内容 |
|---|---|
| 需給調整 | 電力の需要と供給のバランスを取る |
| 省エネルギー | 電力の無駄を減らす |
| 安定供給 | 停電や供給不足を防ぎやすくする |
| 再生可能エネルギー活用 | 太陽光や風力などの変動を吸収しやすくする |
| 見える化 | 電力使用量を把握し、制御に活かす |

例えば、スマートメーターを使うと、電力使用量を細かく把握できます。

```text
スマートメーター
↓
電力使用量を計測
↓
通信で情報を送る
↓
電力需給の調整に活用
```

このテーマは、基本情報技術者試験の「システム戦略」や「情報化と社会」と関係する内容です。公式の出題範囲やシラバスは、[IPA：基本情報技術者試験](https://www.ipa.go.jp/shiken/kubun/fe.html) から確認できます。

## 科目Aでどう出る？

科目Aでは、スマートグリッドの説明として適切なものを選ぶ問題が出題されやすいです。

| 問題文の表現 | 判断 |
|---|---|
| 発電と電力消費を総合的に制御 | スマートグリッド |
| 電力需要を自律的に調整 | スマートグリッド |
| 再生可能エネルギーの活用 | スマートグリッド |
| 安定的な電力供給 | スマートグリッド |
| 電力網に通信・情報処理技術を使う | スマートグリッド |

選択肢では、次のように切り分けます。

| 選択肢の内容 | 該当しやすい用語 |
|---|---|
| 類似症例に基づき健康アドバイスを出す | エキスパートシステム |
| 在宅勤務や多様なワークスタイルを支援する | テレワーク支援システム |
| 自動車内で交通情報や娯楽情報を扱う | カーナビゲーションシステム |
| 電力の需給をICTで制御する | スマートグリッド |

```text
電力 + ICT + 需給調整
→ スマートグリッド
```

### 再生可能エネルギーとの違い

スマートグリッドは、太陽光発電や風力発電そのものではありません。

```text
再生可能エネルギー
→ 電力を作る方法

スマートグリッド
→ 発電と消費の情報を使い、電力網全体を調整する仕組み
```

## どんな場面で使う？

スマートグリッドは、発電量や電力需要が時間によって変化する中で、需給バランスを保つ場面で使われます。

```text
太陽光発電の出力が天気で変わる
↓
電力需要も時間帯で変わる
↓
電力の過不足が起きやすい
↓
ICTで需給を調整する
```

また、電力使用量を見える化することで、利用者側の行動を変え、ピーク時間帯の需要を抑えることにもつながります。

```text
電力使用量を把握する
↓
ピーク時間帯の使用を減らす
↓
電力需要を平準化する
```

## よくある誤解・混同

| 誤解 | 正しい理解 |
|---|---|
| スマートグリッドは健康アドバイスのシステム | 電力網を制御する仕組み |
| スマートグリッドは在宅勤務支援 | 働き方支援ではなく電力需給の調整 |
| スマートグリッドはカーナビの交通情報機能 | 自動車向け情報サービスではない |
| スマートグリッドは再生可能エネルギーそのもの | 再生可能エネルギーを活用しやすくする電力網 |
| スマートグリッドは単なる省エネ家電 | 電力網全体の制御がポイント |

```text
電力の需給調整 → スマートグリッド
健康判断・助言 → エキスパートシステム
多様な働き方 → テレワーク支援
車内情報・交通案内 → カーナビゲーション
```

特に、スマートグリッドは**電力**と**ICTによる制御**がセットで出てくる点を押さえます。

## まとめ（試験直前用）

- スマートグリッドは、ICTで電力の供給と需要を制御する電力網
- 発電量と電力消費を把握し、需給バランスを調整する
- 再生可能エネルギーの活用、安定供給、省エネに関係する
- スマートメーターなどで電力使用量を見える化する
- 健康アドバイス、在宅勤務支援、カーナビとは別物
- 「電力」+「通信・情報処理技術」+「需給調整」が出たらスマートグリッド
