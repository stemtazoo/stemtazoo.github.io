## まず結論

コモディティ化とは、**製品やサービスの機能・品質の差が小さくなり、価格以外で差別化しにくくなること**です。

基本情報技術者試験では、次の表現が決め手になります。

```text
他社製品との差が小さい
価格以外で差別化しにくい
価格競争になりやすい
```

コモディティ化は、新しい製品を生み出す活動ではなく、**市場で製品の違いが見えにくくなった状態**を表します。

## 直感的な説明

ある製品が登場したばかりのころは、メーカーごとに性能や機能の差が大きいことがあります。

しかし、技術が成熟して多くの企業が似た製品を作れるようになると、次のような状態になります。

```text
どの製品も必要な機能を備えている
品質にも大きな差がない
購入者から見ると違いが分かりにくい
```

すると、購入者は価格を重視しやすくなります。

```text
違いが分からない
↓
安い方を選ぶ
↓
価格競争が激しくなる
```

このような状態がコモディティ化です。

## 定義・仕組み

コモディティ化は、製品やサービスが一般化し、企業ごとの特徴が見えにくくなる現象です。

主な背景には次のようなものがあります。

- 技術が成熟し、各社の性能差が小さくなる
- 他社が機能をまねしやすくなる
- 製品の標準化が進む
- 顧客が必要とする基本機能が満たされる

その結果、企業は価格以外の価値を作りにくくなります。

| 用語 | 表すもの | 問題文の手掛かり |
|---|---|---|
| コモディティ化 | 差別化しにくくなった市場の状態 | 品質差が小さい、価格競争 |
| プロダクトイノベーション | 新しい製品・サービスを生み出す革新 | 新製品、新機能、革新的な発明 |
| 破壊的イノベーション | 後発の製品や仕組みが既存市場を変えること | 後発製品、既存市場の衰退・置換 |
| ブルーオーシャン戦略 | 競争の少ない新しい市場を作る戦略 | 新市場、競争を避ける、新しい価値 |

ブルーオーシャン戦略を中心に整理したい場合は、[ブルーオーシャン戦略とは？差別化戦略との違い]({{ '/fe/blue-ocean-strategy/' | relative_url }})もあわせて確認してください。

## 科目Aでどう出る？

科目Aでは、説明文と用語の組合せを選ぶ問題として出題されます。

コモディティ化を疑う表現は次のとおりです。

- 製品の機能や品質の差が小さくなる
- 他社製品との差別化が難しくなる
- 価格以外の競争が難しくなる
- 値下げ競争が起こりやすくなる

試験では、次の切り分けを使います。

```text
違いがなくなる
→ コモディティ化

新しい製品を生み出す
→ プロダクトイノベーション

後発製品が既存市場を置き換える
→ 破壊的イノベーション

競争の少ない新市場を作る
→ ブルーオーシャン戦略
```

特に、「新しい機能を持った製品を作る」という説明だけでブルーオーシャン戦略を選ばないようにします。

ブルーオーシャン戦略では、単なる機能追加ではなく、**新しい価値と市場を作り、既存の競争を避けること**が中心です。

## どんな場面で使う？

企業が自社製品の競争力を考えるとき、コモディティ化は重要な視点になります。

例えば、次のような状況です。

```text
競合製品と性能がほぼ同じ
↓
価格でしか選ばれにくい
↓
利益率が下がる
```

この状態から抜け出すために、企業は次のような方法を考えます。

- 独自機能を追加する
- ブランドやサービスで差別化する
- 新しい顧客層を開拓する
- 新市場を作る

ただし、これらの対応策そのものがコモディティ化ではありません。

```text
コモディティ化
→ 市場の状態

差別化戦略・ブルーオーシャン戦略
→ 状態への対応策
```

## よくある誤解・混同

| 誤解 | 正しい理解 |
|---|---|
| コモディティ化は新製品を開発すること | 新製品の開発はプロダクトイノベーション |
| 後発製品が先発製品を追い出すこと | それは破壊的イノベーションの説明 |
| 他社と違う機能を追加すれば必ずブルーオーシャン | 既存市場での機能差別化だけではブルーオーシャンとは限らない |
| コモディティ化は企業の戦略 | コモディティ化は市場や製品の状態 |
| 価格が安い製品は全てコモディティ | 価格ではなく、差別化の困難さが判断の中心 |

一番重要な切り分けは、**状態なのか、革新なのか、戦略なのか**です。

```text
差がなくなった状態
→ コモディティ化

新しい製品・技術を生む革新
→ イノベーション

競争を避けるための方針
→ ブルーオーシャン戦略
```

## まとめ（試験直前用）

- コモディティ化は、製品やサービスの差が小さくなる状態
- 価格以外で差別化しにくくなり、価格競争が起こりやすい
- 新製品を生み出すのはプロダクトイノベーション
- 後発製品が既存市場を変えるのは破壊的イノベーション
- 競争の少ない新市場を作るのはブルーオーシャン戦略
