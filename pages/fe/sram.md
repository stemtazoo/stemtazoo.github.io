## まず結論

SRAMは、**フリップフロップ回路で情報を保持する、揮発性の半導体メモリ**です。

電源が供給されている間は、DRAMのようなリフレッシュを行わなくても内容を保持できます。基本情報技術者試験では、次の組合せを覚えると選択肢を切りやすくなります。

```text
リフレッシュ不要
＋
高速
＋
ビット単価が高い
＋
キャッシュメモリに使われる
→ SRAM
```

## 直感的な説明

SRAMは、スイッチの状態をそのまま保つ記憶だと考えると分かりやすいです。

```text
スイッチがON
→ 1

スイッチがOFF
→ 0
```

電源が入っている間は、回路がその状態を保ち続けます。そのため、DRAMのように一定間隔で電荷を補い直すリフレッシュは不要です。

ただし、状態を保つために複数のトランジスタを使うので、メモリセルは複雑になり、同じ面積に多くのビットを詰め込みにくくなります。

## 定義・仕組み

SRAMは、Static Random Access Memoryの略です。

「Static」は、電源が供給されている間、記憶内容を保つためのリフレッシュ処理が不要であることを表します。

SRAMは、一般にフリップフロップ回路を使って1ビットの情報を保持します。回路が状態を保つため、高速に読み書きできます。

一方で、1ビットを保持するために複数のトランジスタを使うので、DRAMよりメモリセルが複雑です。そのため、高集積化しにくく、ビット単価も高くなります。

| 特徴 | SRAM |
|---|---|
| 揮発性 | あり。電源を切ると内容が消える |
| リフレッシュ | 不要 |
| 速度 | 高速 |
| 集積度 | DRAMより低くなりやすい |
| ビット単価 | 高い |
| 主な用途 | キャッシュメモリ |

このテーマは、基本情報技術者試験の「コンピュータシステム」や「メモリ」と関係する内容です。公式の出題範囲やシラバスは、[IPA：基本情報技術者試験](https://www.ipa.go.jp/shiken/kubun/fe.html) から確認できます。

## 科目Aでどう出る？

科目Aでは、メモリの特徴を示す記述から、SRAM・DRAM・フラッシュメモリなどを選ぶ問題が出ます。

| 問題文の表現 | 選びたい用語 |
|---|---|
| 高速で、リフレッシュ不要 | SRAM |
| キャッシュメモリに使われる | SRAM |
| ビット単価が高い | SRAM |
| リフレッシュが必要 | DRAM |
| 高集積化しやすく、ビット単価が安い | DRAM |
| 主記憶として使われる | DRAM |
| 電源を切っても内容が残る | ROM・フラッシュメモリなど |
| ブロック単位で消去・書換えする | フラッシュメモリ |

判断するときは、次の順番で見ると整理しやすいです。

```text
1. 電源を切ると消えるか
2. リフレッシュが必要か
3. 主な用途は主記憶かキャッシュか
4. 高速・高価か、高集積・安価か
```

```text
電源断で消える
＋
リフレッシュ不要
＋
高速
＋
キャッシュ
→ SRAM
```

```text
電源断で消える
＋
リフレッシュ必要
＋
高集積・安価
＋
主記憶
→ DRAM
```

DRAMの仕組みやリフレッシュとの関係は、[DRAMとは？リフレッシュが必要な主記憶の仕組み]({{ '/fe/dram/' | relative_url }})で詳しく整理しています。

電源を切っても内容を保持するメモリについては、[フラッシュメモリとは？EEPROM・SRAM・DRAMとの違い]({{ '/fe/flash-memory/' | relative_url }})で確認できます。

## どんな場面で使う？

SRAMは、CPUが頻繁に使うデータを高速に読み書きしたい場面で使われます。

代表的な用途は、CPUと主記憶の速度差を埋めるキャッシュメモリです。

```text
CPU
↓ 高速に使いたいデータ
キャッシュメモリ（SRAM）
↓
主記憶（DRAM）
```

キャッシュメモリには大容量よりも速度が重視されます。そのため、高価でも高速なSRAMが向いています。

一方、主記憶では大容量と価格のバランスが重要なので、高集積化しやすく安価なDRAMが使われます。

```text
高速を優先
→ SRAM

大容量・安価を優先
→ DRAM
```

## よくある誤解・混同

| 混同 | 正しい切り分け |
|---|---|
| SRAMは電源を切っても内容を保持する | SRAMは揮発性なので、電源を切ると内容が消える |
| SRAMはリフレッシュが必要 | リフレッシュ不要なのがSRAM |
| SRAMは主記憶に広く使われる | 主記憶には大容量・安価なDRAMが代表的 |
| SRAMはDRAMより安い | SRAMは高速だが、ビット単価は高い |
| SRAMは高集積化しやすい | 高集積化しやすいのはDRAM |
| ブロック単位で消去するのがSRAM | ブロック単位で消去するのはフラッシュメモリ |

SRAMとDRAMは、どちらも電源を切ると内容が消れる揮発性メモリです。

違いは、リフレッシュの有無、速度、集積度、価格、用途です。

| 比較項目 | SRAM | DRAM |
|---|---|---|
| リフレッシュ | 不要 | 必要 |
| 速度 | 速い | SRAMより遅い |
| 集積度 | 低くなりやすい | 高くしやすい |
| ビット単価 | 高い | 安い |
| 主な用途 | キャッシュメモリ | 主記憶 |

## まとめ（試験直前用）

- SRAMは、フリップフロップ回路で情報を保持する揮発性メモリ
- 電源供給中はリフレッシュが不要
- 高速だが、メモリセルが複雑でビット単価は高い
- キャッシュメモリにはSRAM、主記憶にはDRAM
- 電源を切っても保持するのは、フラッシュメモリなどの不揮発性メモリ
- **高速・リフレッシュ不要・キャッシュならSRAM**
