## まず結論

BPO、SaaS、ホスティングは、どれも外部サービスに関係する用語ですが、任せている対象が違います。

基本情報技術者試験では、次のように切り分けると分かりやすいです。

```text
ホスティング
→ サーバを借りる

SaaS
→ ソフトウェア機能をネット経由で使う

BPO
→ 業務プロセスごと外部に任せる

オフショアアウトソーシング
→ 海外の外部業者に任せる
```

問題文に次のような表現があれば、BPOを疑います。

```text
管理部門やコールセンターなどの業務プロセス全般を、
業務システムの運用などと一体として外部専門業者に委託する
```

## 直感的な説明

違いは、「何を外部に任せているか」で考えると分かりやすいです。

```text
サーバだけ借りる
→ ホスティング

ソフトの機能を使う
→ SaaS

業務そのものを任せる
→ BPO
```

例えば、コールセンター業務を外部企業に任せる場合を考えます。

単にサーバを借りているわけではありません。

単にソフトを使っているだけでもありません。

問い合わせ受付、対応、記録、業務システムの運用など、業務の流れごと任せているならBPOです。

## 定義・仕組み

BPOは、Business Process Outsourcing の略です。

企業の業務プロセスの一部、または全体を、外部の専門業者に委託することを指します。

| 用語 | 何を使う・任せる？ | 例 |
|---|---|---|
| ホスティング | サーバ資源 | Webサーバを借りる |
| SaaS | ソフトウェア機能 | メール、会計、CRMなどをネット経由で使う |
| BPO | 業務プロセス | コールセンター、給与計算、経理処理などを委託する |
| オフショアアウトソーシング | 海外の外部業者 | 海外企業へ開発や運用を委託する |

BPOでは、単なる作業の一部ではなく、業務プロセスとしてまとまった範囲を外部に任せる点が特徴です。

```text
問い合わせを受ける
↓
内容を記録する
↓
対応する
↓
結果を報告する
```

このような業務の流れを外部に任せると、BPOと呼ばれます。

このテーマは、基本情報技術者試験の「システム戦略」や「業務プロセス」と関係する内容です。公式の出題範囲やシラバスは、[IPA：基本情報技術者試験](https://www.ipa.go.jp/shiken/kubun/fe.html) から確認できます。

## 科目Aでどう出る？

科目Aでは、説明文を読んで該当する用語を選ぶ問題として出題されやすいです。

次の表現があれば、BPOを疑います。

| 問題文の表現 | 判断 |
|---|---|
| 業務プロセスを外部委託 | BPO |
| コールセンターや管理部門の業務を外部へ任せる | BPO |
| 業務システム運用と一体で委託する | BPO |
| 自社の本来業務に集中するために外部化 | BPOやアウトソーシング |

一方で、次の表現なら別の用語です。

| 問題文の表現 | 該当しやすい用語 |
|---|---|
| サーバの処理能力や記憶容量を借りる | ホスティング |
| ソフトウェアの機能をネットワーク経由で利用 | SaaS |
| 海外の外部業者へ委託 | オフショアアウトソーシング |
| 人件費の安い派遣社員を使う | BPOそのものではない |

試験では、次のように切り分けます。

```text
サーバを借りる → ホスティング
ソフトを使う → SaaS
業務プロセスを任せる → BPO
人を安く使う → BPOとは限らない
```

## 科目Bでどう使う？

科目Bでは、業務改善や外部委託の判断として出てくる可能性があります。

BPOを使う目的は、単なるコスト削減だけではありません。

```text
専門業者のノウハウを使う
↓
業務を効率化する
↓
自社は本来の業務に集中する
```

ただし、外部に任せる範囲が広くなるため、注意点もあります。

```text
委託先との責任分担
サービス品質
情報セキュリティ
業務知識の社内蓄積
```

科目Bでは、BPOを見たら、外部委託のメリットだけでなく、委託先管理の視点も少し持っておくと読みやすくなります。

## よくある誤解・混同

BPO、SaaS、ホスティングでよくある誤解は、外部サービスをすべて同じものとして見てしまうことです。

| 誤解 | 正しい理解 |
|---|---|
| BPOはサーバを借りること | サーバ利用はホスティング |
| BPOはソフトをネットで使うこと | ソフト利用はSaaS |
| SaaSは業務プロセス全体の委託 | SaaSはソフトウェア機能の利用 |
| ホスティングは業務委託 | ホスティングはサーバ資源の利用 |
| BPOは単に安い人材を使うこと | BPOは業務プロセスを外部委託すること |

試験では、次の切り分けを残しておくと強いです。

```text
資源を借りる → ホスティング
機能を使う → SaaS
業務を任せる → BPO
海外へ任せる → オフショアアウトソーシング
```

特に、BPOは **業務プロセス** という言葉が出る点を押さえます。

## まとめ（試験直前用）

- BPOは、業務プロセスを外部専門業者に委託すること
- SaaSは、ソフトウェア機能をネットワーク経由で利用すること
- ホスティングは、サーバの処理能力や記憶容量などを借りること
- オフショアアウトソーシングは、海外の外部業者へ委託すること
- サーバならホスティング、ソフトならSaaS、業務ならBPO
- BPOは単なる人件費削減ではなく、業務プロセス単位で任せる点が重要
