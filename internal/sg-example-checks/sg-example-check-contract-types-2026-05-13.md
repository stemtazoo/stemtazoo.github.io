# SG例題対応チェック（請負契約の請負人の義務）

- 作成日: 2026-05-13
- 対象: 民法上の請負契約で、請負人の義務を問う3記述問題
- 判定: 既存記事に追記推奨（実施済み）

## 1. 例題の論点

- 問われているテーマ:
  - 請負契約における請負人の義務（完成義務、契約不適合責任、作業主体）
- 正解に必要な知識:
  - 完成義務はある
  - 欠陥への責任は負う
  - ただし、原則として全作業を請負人自身が行う義務まではない
- 誤答選択肢を切るポイント:
  - 「完成責任」と「作業主体固定義務」を混同しない
- 混同しやすい用語:
  - 請負契約 / 準委任契約
  - 完成義務 / 善管注意義務

## 4. 理由

今回問題は、請負契約の基本（完成責任）に加え、「請負人が必ず自分で全作業を行うか」をひっかけるタイプです。
既存の請負vs準委任記事を軸に、作業主体に関する誤解を明示すると、同型問題を安定して切れます。

## 5. 具体的な対応案

### 既存記事に追記する場合

- 追記先の記事:
  - `pages/sg/contract-types.md`
- 追記する見出し:
  - 「請負人は自分の手で全部行う義務があるか」
- 追記内容の要約:
  - 請負は完成義務が本質であり、作業主体固定義務とは別論点
