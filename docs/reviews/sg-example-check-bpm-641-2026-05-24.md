# 例題対応チェック結果（第641問: BPM）

## 1. 例題の論点

- 問われているテーマ：
  - BPM（Business Process Management）の定義と、ERP/CRM/SCMとの切り分け
- 正解に必要な知識：
  - BPMは業務プロセスを対象に、分析・設計・実行・評価・改善のサイクルを継続的に回す管理手法
  - 継続的改善がキーワード（BPRのような一度限りの抜本改革とは異なる）
- 誤答選択肢を切るポイント：
  - 企業全体の経営資源の一元管理はERP
  - 顧客接点業務（販売〜保守〜問合せ等）の一貫管理はCRM
  - 調達〜生産〜物流〜販売の連携最適化はSCM
- 混同しやすい用語：
  - BPM vs BPR（継続改善か、抜本再設計か）
  - BPM vs ERP（企業資源統合管理）
  - BPM vs CRM（顧客関係管理）
  - BPM vs SCM（供給連鎖最適化）
