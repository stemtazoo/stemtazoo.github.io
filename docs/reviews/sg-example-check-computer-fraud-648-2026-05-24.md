# 例題対応チェック結果（第648問: 電子計算機使用詐欺罪）

## 1. 例題の論点

- 問われているテーマ：
  - 刑法上の「電子計算機使用詐欺罪」が適用される行為の識別
- 正解に必要な知識：
  - コンピュータに虚偽情報・不正指令を与え、財産上不法の利益を得る（または他人に得させる）行為が中心
  - 典型例は不正振込・不正送金など、金融システムの処理を悪用して財産的利益を得るケース
- 誤答選択肢を切るポイント：
  - ねずみ講の開設は無限連鎖講防止法の文脈（詐欺罪が関係する場合は別途）
  - 実際より良品と誤認させる表示は景品表示法の文脈
  - Web改ざんで企業業務を妨害する文脈は電子計算機損壊等業務妨害罪
- 混同しやすい用語：
  - 電子計算機使用詐欺罪 vs 詐欺罪（人をだます）
  - 電子計算機使用詐欺罪 vs 電子計算機損壊等業務妨害罪（業務妨害）
  - 刑法犯罪類型 vs 行政法規（景表法、無限連鎖講防止法）

## 4. 理由

本例題は、
- 「コンピュータ処理の悪用」
- 「財産的利益の取得」
を満たす行為（不正振込・送金）を選べるかが核心。
`computer-fraud.md` で中心論点が明示され、さらに
`denshi-keisanki-sonkai-gyomu-bogai.md` で業務妨害との比較が可能なため、
新規記事なしで対応可能。
