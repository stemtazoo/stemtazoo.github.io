# 例題対応チェック結果（第633問: 通信傍受法）

## 1. 例題の論点

- 問われているテーマ：
  - 通信傍受法（犯罪捜査のための通信傍受に関する法律）の制度要件と、電気通信事業者の協力義務の理解
- 正解に必要な知識：
  - 傍受は事業者判断で実施できず、裁判官による令状手続が必要
  - 傍受期間には上限があり、無制限ではない（通算30日上限）
  - 通信事業者等には、正当理由のない協力拒否を認めない趣旨の協力義務規定がある
- 誤答選択肢を切るポイント：
  - 通信傍受法制定時に電気通信事業法の通信の秘密条文が変更されたという理解は誤り
  - 犯罪予見だけで事業者判断の傍受は不可（令状主義）
  - 傍受期間無制限は誤り（期間上限あり）
- 混同しやすい用語：
  - 通信傍受法 vs 電気通信事業法（通信の秘密）
  - 捜査機関の令状手続 vs 事業者の自主判断
  - 協力義務の存在 vs 任意協力の誤解

## 4. 理由

今回の例題は、通信傍受法固有の制度要件（令状・期間上限・協力義務）を直接問う。
既存記事は電気通信事業法など周辺法の説明が中心で、
「通信傍受法そのもの」を1ページで切り分ける学習導線が不足している。
そのため、既存記事追記だけよりも、通信傍受法を独立記事で整理した方が、
SG試験の選択肢除外（事業者判断で傍受可／期間無制限などの誤り判定）に直結する。
