# SG例題対応チェック（サイドチャネル攻撃・タイミング攻撃）

## 例題の要点
- 論点: サイドチャネル攻撃の一種であるタイミング攻撃の対策を選ぶ。
- 正解軸: 「処理時間の差を出さない（一定時間化）」を選べるか。
- 誤答排除: フォルトアタック対策・電力解析対策・改ざん対策を区別できるか。

## 既存記事確認メモ
- `pages/sg/side-channel-attack.md` は、処理時間・電力・電磁波を観測対象として明示し、対策に「処理時間が秘密情報に依存しないようにする」を含む。
- `pages/sg/crypto-attacks-summary.md` は、攻撃者が何を材料にするかで切る方針を表で整理。
- `pages/sg/related-plaintext-attack.md` は、関連平文攻撃との切り分けを補強。
- `pages/sg/aes.md` は、AES本体とサイドチャネル攻撃の混同防止に有効。

## 判定
- 判定: **既存記事で十分（ただし直前確認用としては side-channel 記事を最優先導線にするのが望ましい）**

## 補足
- 例題で出る「フォルトアタック」「電力解析攻撃」をサイドチャネル記事内の比較表でさらに明確化すると、誤答排除がより速くなる。
