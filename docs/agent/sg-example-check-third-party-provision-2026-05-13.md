# SG例題対応チェック（個人情報の第三者提供と本人同意）

- 作成日: 2026-05-13
- 対象: 個人情報保護法における第三者提供の同意要否（4択）
- 判定: 新規記事作成推奨（実施済み）

## 1. 例題の論点

- 問われているテーマ:
  - 個人データ第三者提供の原則同意と4例外
- 正解に必要な知識:
  - 例外に該当しなければ本人同意が必要
  - 「本部要請」だけでは例外にならない
- 誤答選択肢を切るポイント:
  - 法令根拠（令状）
  - 公衆衛生・児童健全育成
  - 生命身体財産保護＋同意困難
  - 公的機関の法定事務協力
- 混同しやすい用語:
  - 要配慮個人情報の同意要件
  - 匿名加工情報の第三者提供
  - 取引上必要性と法定例外

## 4. 理由

今回の設問は「個人情報法の全体説明」ではなく、第三者提供の同意要否を4例外で瞬時に切る力を問うタイプです。
既存記事だけだと、要配慮個人情報や匿名加工情報の論点へ意識が分散しやすいため、
同意要否に特化した1記事の新設が最短と判断しました。
