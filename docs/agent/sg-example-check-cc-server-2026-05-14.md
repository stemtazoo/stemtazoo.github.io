## 論点
- C&Cサーバは感染端末（ボット）へ命令を出す指令役。
- ボットネット（感染端末群）とC&Cサーバ（司令塔）の役割分離。
- 誤答切り分け: 防御機器、証明書発行、Webコンテンツ管理など別領域を除外。
