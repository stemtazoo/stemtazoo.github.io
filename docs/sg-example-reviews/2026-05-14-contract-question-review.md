# SG例題レビュー（2026-05-14）

## 例題
請負契約の下で、自己の雇用する労働者を契約先の事業所などで働かせる場合、適切なものはどれか。

## 論点整理
- テーマ: 請負契約と派遣契約の切り分け（特に指揮命令関係と雇用関係）。
- 正解判断に必要な知識:
  - 請負では、労働者は請負企業（雇用主）の指揮命令下で働く。
  - 契約先（発注者）と労働者の間に雇用関係は生じない。
  - 労働条件（勤務時間・休日など）は雇用主側が管理する。
- 誤答除外に必要な知識:
  - 「契約先が労働条件を管理する」は派遣の発想であり請負では不適切。
  - 「契約先の指揮命令で働く」は請負では不適切（偽装請負リスク）。
  - 「契約先との間にも雇用関係が生じる」は誤り。
- 混同しやすい用語:
  - 請負契約 / 派遣契約 / 準委任契約
  - 指揮命令関係 / 雇用関係 / 労働条件の管理主体
- ひっかけ:
  - 「契約先の事業所で働く」事実だけで、契約先が指揮命令できると誤認させる。
